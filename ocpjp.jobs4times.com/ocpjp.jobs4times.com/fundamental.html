Question : 26

Which are valid declarations ? (Choose all that apply)
  1. int $x;
  2. int 123;
  3. int _123;
  4. int #dim;
  5. int %percent;
  6. int *divide;
  7. int central_sales_region_Summer_2005_gross_sales;

Answer : A, C, G

  • A,C,G are legal identifiers.
  • B is incorrect because an identifier can't start with a digit.
  • D,E,F are incorrect because identifiers must start with $, _, or a letter.

Question : 29

Which are legal declarations ? (Choose all that apply)
  1. short x[];
  2. short[] y;
  3. short[5] x2;
  4. short z2[5];
  5. short[] z[][];
  6. short[] y2=[5];

Answer : A, B, E

  • A, B and E are correct array declarations; E is a three dimensional array.
  • C, D and F are incorrect, you can't include the size of your array in a declaration unless you also instantiate the array object.F uses invalid instantiation syntax.
